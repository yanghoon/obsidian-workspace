* Redis Eviction Policy
```bash
maxmemory 100mb # 3gb (32bit), 0 (64bit)

maxmemory-eviction <policy>
# noeviction (wirte fail, set to master when have replicas)
#
# volatile := expire true
# lru := approximate
#
# allkeys-lru|lfu
# allkeys-random
#
# volatile-lru|lfu
# volatile-random
# volatile-ttl (shortest ttl)
```
