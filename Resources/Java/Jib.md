>Jib builds optimized Docker and [OCI](https://github.com/opencontainers/image-spec) images for your Java applications without a Docker daemon

## Usage

```groovy
// build.gradle
plugins {
  id 'com.google.cloud.tools.jib' version '3.4.0'
}
```
